# Bitcoin Transaction Visualisation and Anomaly Detection

A final-year project that explores real-time Bitcoin transaction visualisation and unsupervised anomaly detection.

## Project Objectives

 Collect and process Bitcoin transaction data.
 Extract meaningful transaction features.
 Detect unusual transactions using anomaly detection methods.
 Visualise transaction activity and anomaly scores in an interactive web application.
 Evaluate the accuracy, performance, and limitations of the system.

## Planned Technology

 Python and FastAPI
 Pandas and scikit-learn
 React and JavaScript
 Apache ECharts
 SQLite and WebSocket

## Project Structure

 `backend/` - data collection, feature extraction, API, and anomaly detection
 `frontend/` - interactive web visualisation
 `docs/` - project planning and research documentation
